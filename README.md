# go-rest

Just a simple go rest with PostgreSQL.
Very easy to config.

## Installation
1. Clone repo
    ```bash
    $ git clone https://github.com/geshtng/go-rest
    ```

2. Migrate database. Use my sample database in `sample_db` folder
3. Run
    ```bash
    $ make run
    ```
    or
    ```bash
    $ go run main.go
    ```
